What is Lung Cancer AI?

Lung Cancer AI is a web-based medical application that uses advanced artificial intelligence and deep learning to assist doctors and healthcare professionals in classifying lung scan images into three distinct categories:

  • Normal: No suspicious findings detected
  • Benign: Non-cancerous abnormalities detected
  • Malignant: Potentially cancerous findings detected

This classification helps medical professionals quickly identify suspicious images that require further investigation, potentially leading to earlier detection and improved patient outcomes.

How the AI Works

Model Architecture

  • Utilizes an improved Convolutional Neural Network (CNN) with residual connections and attention mechanisms
  • Features progressive blocks with increasing filter sizes (64, 128, 256, 512)
  • Incorporates residual (skip) connections to maintain gradient flow and improve training
  • Implements channel-wise attention mechanisms that help the model focus on relevant features
  • Uses SELU activation functions for improved performance
  • Employs spatial dropout and batch normalization for effective regularization
  • Finalizes with global average pooling and dense layers for classification

Data and Training

  • Dataset: Composed of chest images classified into benign, malignant, and normal cases
  • Data Augmentation: Enhanced by extensive augmentation techniques including:
    • 360Â° rotation
    • Horizontal and vertical flipping
    • Width and height shifts (up to 20%)
    • Zoom adjustments (up to 20%)
    • Brightness variations (80% to 120%)
  • Class Balancing: Uses SMOTE (Synthetic Minority Over-sampling Technique) to address class imbalance
  • Advanced Inference: Implements Test-Time Augmentation (TTA) for robust predictions
  • Training Details:
    • Optimizer: Adam with learning rate of 1e-4
    • Loss: Categorical Crossentropy
    • Metrics: Accuracy
    • Callbacks: Early Stopping, Reduce Learning Rate on Plateau, Model Checkpoint, TensorBoard

Input Requirements

The AI model expects input images with the following specifications:

  • Format: JPEG, PNG, JPG
  • Dimensions: Resized automatically to 224x224 pixels
  • Color Space: RGB (color images)
  • Normalization: Pixel values scaled between 0 and 1
Note: For optimal results, images should be clear, properly oriented lung scans without excessive noise or artifacts.

API & Prediction Workflow

The application provides a simple RESTful API for integration:

  • Endpoint: /predict
  • Method: POST
  • Input: Image file via form-data
  • Output: JSON response with:
    • status: "success" or "error"
    • prediction: Class index (0: Benign, 1: Malignant, 2: Normal)
    • confidence: Array of confidence scores for each class

Additionally, a /health endpoint verifies that the API is running correctly with a GET request.

Limitations and Notes

Important: This AI tool is intended as an assistive technology and does not replace clinical judgment.

  • Predictions should always be confirmed with clinical evaluation and additional diagnostic testing
  • The model's predictions depend heavily on the quality of input images
  • The system may occasionally produce false positives or false negatives
  • Regular model updates and retraining are necessary to maintain and improve performance
  • This tool is designed to assist medical professionals and should not be used for self-diagnosis
